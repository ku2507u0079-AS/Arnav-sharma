A full-stack event registration application built with HTML, CSS, JavaScript, and SQL. This app allows users to register for events, view event details, and manage registrations. The project demonstrates front-end design, client-side interactivity, and database integration for storing and retrieving user registration data.

Features:

User-friendly registration form
Event listing and details page
SQL database for storing registration data
Responsive design for desktop and mobile
Basic client-side validation with JavaScript

Tech Stack:

Front-end: HTML, CSS, JavaScript
Back-end/Database: SQL

Usage:

Clone the repository
Set up the SQL database and import the provided schema
Open the application in a web browser to start registering for events
